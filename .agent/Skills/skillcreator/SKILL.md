# Skill Creator for RotuloLab

This skill is designed to assist in the development and automation of the RotuloLab project. It focuses on creating new skills, components, and tools specifically tailored for nutraceutical label design.

## Responsibilities
- Create and update project-specific skills.
- Automate the generation of design components (Canva-style).
- Ensure compliance with regulatory standards for supplement labels.
- Facilitate the creation of PDF exports for printing.

## Usage
Use this skill when you need to expand the capabilities of the RotuloLab agent or create new specialized workflows for label creation.
